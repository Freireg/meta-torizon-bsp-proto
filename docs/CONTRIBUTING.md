# How to Contribute

The contributing guide is maintained in one place for both Torizon OS layers
(`meta-torizon` and `meta-torizon-bsp`):

**https://freireg.github.io/contributing/**

In short: fork, branch, sign off your commits (`git commit -s`), and open a pull
request against the default branch. Reviewers are assigned automatically from
[`docs/CODEOWNERS`](./CODEOWNERS). See the link above for the full guidelines.
